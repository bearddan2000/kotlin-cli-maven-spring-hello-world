# kotlin-cli-maven-spring-hello-world

## Description
A POC for kotlin spring maven build.

## Tech stack
- kotlin
- maven

## Docker stack
- openjdk:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
